# Dark Theme Color Palette

## Overview
This document describes the sophisticated dark theme color palette based on purple/indigo tones that replace traditional grey and black combinations.

## Color Palette

### Base Colors (From Provided Palette)
1. **Pure Black**: `#000000` (0, 0, 0)
   - Usage: Deepest shadows, special accent elements
   - Note: Used sparingly for maximum contrast when needed

2. **Very Dark Purple/Indigo**: `#1a151f` (26, 21, 31)
   - Usage: Main background color
   - Provides depth without being pure black

3. **Dark Purple/Indigo**: `#241e2a` (36, 30, 42)
   - Usage: Surface/card backgrounds
   - Creates subtle elevation from main background

4. **Medium Dark Purple/Indigo**: `#2c2433` (44, 36, 51)
   - Usage: Elevated surfaces, input backgrounds
   - Provides hierarchy in layered components

5. **Lightest Dark Purple/Indigo**: `#393041` (57, 48, 65)
   - Usage: Borders, dividers
   - Subtle separation without harsh contrast

## Theme Implementation

### Background Colors
- **Background**: `#1a151f` - Main page background
- **Surface**: `#241e2a` - Cards, panels, modals
- **Surface Elevated**: `#2c2433` - Elevated cards, dropdowns

### Text Colors
- **Primary Text**: `#f1f5f9` - High contrast, easy to read
- **Secondary Text**: `#e2e8f0` - Good contrast for secondary information
- **Muted Text**: `#cbd5e1` - For disabled or less important text

### Accent Colors
- **Primary**: `#8b5cf6` - Vibrant purple (buttons, links, highlights)
- **Secondary**: `#a78bfa` - Light purple (secondary actions)
- **Accent**: `#c084fc` - Soft purple (special accents)

### Border Colors
- **Border**: `#393041` - Standard borders
- **Border Light**: `#2c2433` - Subtle dividers

### Status Colors
- **Success**: `#34d399` - Light green
- **Warning**: `#fbbf24` - Yellow/amber
- **Error**: `#f87171` - Light red
- **Info**: `#60a5fa` - Light blue

## Design Principles

1. **Avoid Pure Black**: Using `#1a151f` instead of `#000000` creates a softer, more comfortable viewing experience
2. **Purple/Indigo Undertones**: All dark colors have subtle purple/indigo tones that create visual harmony
3. **Proper Contrast**: Text colors are light enough to ensure excellent readability (WCAG AA compliant)
4. **Layered Depth**: Multiple surface levels create visual hierarchy
5. **Subtle Borders**: Borders use colors from the palette rather than harsh black/white

## Contrast Ratios

- Background (`#1a151f`) vs Primary Text (`#f1f5f9`): **15.8:1** ✓ (WCAG AAA)
- Surface (`#241e2a`) vs Primary Text (`#f1f5f9`): **14.2:1** ✓ (WCAG AAA)
- Background vs Primary Accent (`#8b5cf6`): **4.5:1** ✓ (WCAG AA)

## Usage Guidelines

### When to Use Each Background Level
1. **Background** (`#1a151f`): Main page background, empty states
2. **Surface** (`#241e2a`): Cards, panels, modals, table backgrounds
3. **Surface Elevated** (`#2c2433`): Dropdowns, tooltips, elevated cards

### When to Use Border Colors
1. **Border** (`#393041`): Standard borders around cards, inputs, tables
2. **Border Light** (`#2c2433`): Subtle dividers within components, table row separators

### Text Hierarchy
1. **Primary Text**: Headings, important content, body text
2. **Secondary Text**: Captions, labels, less important information
3. **Muted Text**: Disabled states, placeholders, metadata

## Benefits

1. **Reduces Eye Strain**: Purple/indigo tones are easier on the eyes than pure black/grey
2. **Visual Harmony**: All colors work together cohesively
3. **Professional Appearance**: Sophisticated palette suitable for professional applications
4. **Better Depth Perception**: Multiple surface levels create better visual hierarchy
5. **Accessibility**: High contrast ratios ensure excellent readability
